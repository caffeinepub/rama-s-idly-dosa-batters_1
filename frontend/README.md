# Rama's Home Foods — Idly & Dosa Batters

A single-page website for Rama's Home Foods, built on the Internet Computer (ICP) using React + TypeScript + Tailwind CSS.

---

## Prerequisites

- [DFX SDK](https://internetcomputer.org/docs/current/developer-docs/setup/install/) installed (`dfx --version` ≥ 0.15)
- An ICP identity with cycles: `dfx identity whoami`
- Sufficient cycles in your cycles wallet (≥ 1 TC recommended for first deploy)
